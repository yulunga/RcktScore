# RcktScore V2

Version 2 is structured as:

- `frontend/` for the React client deployed with Amplify
- `backend/` for AWS Lambda functions, shared Supabase logic, and SAM config
- `infrastructure/` for deployment configuration notes and build specs
- `version1/` for the original Flask application kept as reference

See [frontend](./frontend), [backend](./backend), and [infrastructure](./infrastructure) for the new implementation.
